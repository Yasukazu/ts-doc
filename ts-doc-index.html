Window, Navigator and Document

Javascript can read and write HTML Document. And also get information about the Window rendered by the web browser. And also information about the web browser itself(Navigator)

Document interface:

  • Read from Document:

    Javascript can get a certain location in the document like this:

    const win_nav_docObj = document.querySelector('#win-nav-doc');

  • Write into Document:

    Javascript can write into the document like this:

    const_win_nav_docObj.textContent = 'The Replaced text';
Such Document APIs are like these(HTMLElement in MDN web docs

Window

Read viewport width

To get the width into a variable 'viewportWidth' as int:

let viewportWidth = window.innerWidth;
Replace 'innerWidth' with 'innerHeight' to get the height.

  • Navigator

    Supported Javascript version depends on Navigator / aka: 'user agent' / aka: 'web browser'(like: Firefox, Chrome, Edge, Safari)).

    Precise appearance and dynamic parts of a web page need to be configured according to the navigator.

    const userAgent = window.navigator.userAgent;
