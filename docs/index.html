Document / Window / Navigator

Javascript can read and write HTML Document. And also get information about the Window rendered by the web browser. And also information about the web browser itself(Navigator)

Document interface:
Read from Document:
Javascript can get a certain location in the document using CSS selector:
const win_nav_docObj = document.querySelector('#win-nav-doc');
Write into Document:
Javascript can write into the document like this:
const_win_nav_docObj.textContent = 'The Replaced text';
Such Document APIs are like these(HTMLElement in MDN web docs
Window interface:
Read viewport width
To get the width into a variable 'viewportWidth' as int:
let viewportWidth = window.innerWidth;
Read viewport height
To get the height into a variable 'viewportHeight' as int:
let viewportHeight = window.innerHeight;
Such Document APIs are like these(Window interface in MDN web docs)
Navigator interface:
User agent:
Supported Javascript version depends on 'user agent' that is 'web browser'(like: Firefox, Chrome, Edge, Safari)).
Ever before, the precise appearance needed to be configured according to the browser rendering engine. Following code gets 'user agent'(web browser) as a string(text):
const userAgent = window.navigator.userAgent;
User language:
The language of browser user interface is useful to fit with it's proper webpage language.
const userLanguage = window.navigator.language;
Language tag before hyphen is like: IETF language tag(Wikipedia)
Document Object Model

In Javascript, 'Document Object Model' is represented as Document type interface. It is called as 'DOM'.

DOM is a tree structure interface corresponds with HTML structure.
In a JavaScript code, the document model is available as the 'document' object.

To get its property:
document.prop
To call its function with parameters:
document.func(param1, param2)
Basic DOM operation

How to operate an HTML document

Create / Append / Move / Remove a node
Create a new element node with a tag:
(create a node object by create-element method call with a parameter(tag))Script:
let divElem = document.createElement('div');
Modify the element:
(JavaScript:) divElem.className = "greeting"; divElem.innerHTML = "He-lo, What's going on here?";
Append the element into the document:
(JavaScript:) document.body.append(divElem);
Append the element node to other location cause 'move', resulting old position one is automatically removed. If neeed to duplicate the element node, use cloneNode(bool) method. If the bool is true then deep copy else no child copy. In case to remove the element node, use remove() method of the node itself.
Event customization

Assigns a function to an event handler(like onclick) of a with-event-handler element(like input) of the DOM object(that is document):

(HTML):
<input id="btn_elem" type="button" value="Click me">
(script):
const btn_elem = document.querySelector("#btn-elem");
const event_org_txt = event_divObj.textContent;
let btn_counter = 0;
btn_elem.onclick = function() { btn_counter += 1;
event_divObj.textContent = `${event_org_txt} + ${btn_counter}`; }

How to get a CSS variable

Use getComputedStyle and getPropertyValue:

const hdr_h = getComputedStyle(document.documentElement).getPropertyValue('--hdr-h');
Set a CSS variable
document.querySelector('body').style.setProperty(nav_h_str, new_nav_h_str);
